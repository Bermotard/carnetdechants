# carnetdechants
Un carnet de champs accessible par une GitHub page.
Le carnet de chant est créé par un autre projet avec la structure suivante:

carnetdechant_export/
├── index.html                  # Interface de lecture universelle

└── data/                       # Conteneur des données d'affichage

    ├── index.json              # Liste ordonnée des carnets et de leurs chants
    
    └── nomducahier1/            # Dossier spécifique à un carnet
    
        ├── pointeur vers chant_1.md          # pointeur vers Chant converti en Markdown
        
        └── pointeur vers chant_2.md          # pointeur versChant converti en Markdown
        
        └── ...
        
    └── nomducahier2/            # Dossier spécifique à un carnet
    
        ├── pointeur vers chant_3.md          # pointeur vers Chant converti en Markdown
        
        └── pointeur vers chant_2.md          # ponteur Chant converti en Markdown
        
        └── 
   
   ├── chants
   
   └── chant_1.md
   
   └── chant_2.md
   
   └── chant_3
   
   └── ....
